# mbview

Watch MBTiles in your localhost

![demo](https://raw.githubusercontent.com/mapbox/mbview/master/demo.gif)

```bash
% npm install
% node cli.js --port 9000 ~/roads.mbtiles ~/taco-places.mbtiles
```

Global install

```bash
% npm install -g mbview
% mbview
```
